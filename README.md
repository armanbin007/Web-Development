## 📖 About

This repository documents my progress learning web development, step by step. Instead of scattering notes and projects across random folders, everything here follows a clear numbered structure — so you (or anyone else browsing) can see exactly how the skills build on each other, from static pages to dynamic, full-stack applications.

## 🗂️ Repository Structure

| Folder | Focus | Description |
|---|---|---|
| [`1 HTML`](./1%20HTML) | Markup | Semantic HTML, page structure, forms, and accessibility basics |
| [`2 CSS`](./2%20CSS) | Styling | Layouts, Flexbox/Grid, responsive design, and styling fundamentals |
| [`3.1 Project NF`](./3.1%20Project%20NF) | Project | A practice build applying core HTML & CSS concepts |
| [`4 JavaScript & NodeJs`](./4%20JavaScript%20%26%20NodeJs) | Logic | Core JavaScript (DOM, events, ES6+) and an introduction to Node.js |
| [`4.1 Project Spotify`](./4.1%20Project%20Spotify) | Project | A Spotify-inspired UI build combining HTML, CSS, and JavaScript |
| [`5 Backend Node.js Npm`](./5%20Backend%20Node.js%20Npm) | Backend | Server-side development with Node.js, npm packages, and EJS templating |
| [`6 React`](./6%20React) | Frontend Framework | Component-based UI development using React |

> 📌 Folder numbers indicate the recommended learning order — start at `1` and work your way up.

## 🛠️ Tech Stack

- **Markup & Styling:** HTML5, CSS3
- **Scripting:** JavaScript (ES6+)
- **Runtime & Templating:** Node.js, npm, EJS
- **Frontend Framework:** React

## 🚀 Getting Started

Clone the repo to explore or run any of the projects locally:

```bash
git clone https://github.com/armanbin007/Web-Development.git
cd Web-Development
```

Then navigate into the folder you're interested in:

```bash
cd "4.1 Project Spotify"
```

For folders involving Node.js or React, install dependencies before running:

```bash
npm install
npm start
```
